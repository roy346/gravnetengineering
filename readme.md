# Gravnet Engineering Website

A professional, responsive landing page for Gravnet Engineering, specializing in Small Business and Home Office IT infrastructure.

## 🛠 Features
- Responsive design (Mobile & Desktop friendly)
- Clean, modern IT-focused aesthetic
- Service overview and contact sections
- Externalized CSS for easy styling updates

## 📂 Project Structure
- `index.html`: The main structure and content of the site.
- `style.css`: The visual styling and layout rules.

## 🚀 How to View
Simply open `index.html` in any web browser.
